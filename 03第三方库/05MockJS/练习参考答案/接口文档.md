# 获取购物车数据

请求路径：/api/cart

请求方法：GET

响应结果示例：

```json
{
  "code": 0, // 无错误
  "msg": "", // 错误消息
  "data": [
    // 主体数据
    {
      "productName": "iphone12", // 商品名称
      "productUrl": "http://img.com/iphone12", // 商品图片url地址
      "unitPrice": 8888, // 商品单价
      "count": 3 // 购物数量
    }
  ]
}
```
